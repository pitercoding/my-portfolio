04

Zoologic

Zoologic is a Java project in development that simulates a classification and management system for vertebrate animals of the phylum Chordata. Created for educational purposes by a Java student, the goal is to apply Object-Oriented Programming (OOP) concepts, package organization, use of Enums, encapsulation, inheritance, polymorphism, and coding best practices. Each animal class has specific attributes and its own methods that override common behaviors from the superclass Chordata. The system allows registering and displaying detailed information about different classes of animals:

  • Mammals
  • Birds
  • Reptiles
  • Fishes
  • Amphibians

Java
